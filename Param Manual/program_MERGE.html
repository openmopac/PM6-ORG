PARAM Utilities: Program MERGE

 

MERGE combines survey tables of heats of formation, dipoles, etc., for various methods, to create HTML documents suitable for posting on the internet.  The files created by MERGE allow easy comparison of different methods. MERGE should only be run when the appropriate survey tables exist.  That is, MERGE should be run after:

(a) Method development is complete, or almost complete.
(b) Surveys have been done of the new method and appropriate older methods

How to run MERGE

First, run surveys of the appropriate methods.  Name each survey after the method, so if AM1 is to be surveyed, the data set used in the survey should be called am1.dat or AM1.dat. 
At the DOS prompt, go to the folder that contains the survey files.  These files will have names that end in ".geos", ".heats", ".ips", ".dips".
Assuming the surveys available are AM1, PM3, PM6, and new, use the command:
C:> MERGE AM1 PM3 PM6 NEW

MERGE is very forgiving - if some data are missing, it will fill the appropriate cell with a space or a "-" symbol.

References will be formatted to make them easy to read.
