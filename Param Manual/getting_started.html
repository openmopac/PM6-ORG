Getting started using PARAM

Download and Unpacking

Download the GitHub repository https://github.com/openmopac/param   PARAM should be on a Windows machine, an appropriate location would be "C:\Users\yourname" another would be any local drive that would be suitable, for example if you want drive M:\ to be used, clone the repository in M:\  

Setting up the PARAM folder

All the instructions for preparing a full working copy of PARAM are given in the file README.txt in the top folder of PARAM.
