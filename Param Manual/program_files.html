PARAM Utility FILES

 When program FILES is run, a list of reference data set file names is constructed, along with information on each file.

Procedure for running FILES

FILES should only be used in a folder that contains a set of MOPAC reference data files.

FILES can only be run in a console window.  At the DOS prompt, enter the command files, followed by a return.

Files generated by FILES

files.out

files.out is used by other and, only, etc. It should not be edited - if it needs to be updated, re-run FILES.
Files.out contains a summary of the database, followed by a list of files, one line per file.  Examples of two typical line would be:

 H2O                water                                                                                 -57.8+  1.9+ 12.6+ H  O
 C6H6               benzener                                                                                           YES   H  C

 The data on this line can be read as follows:

Datum  

Description
H2O   The formula for the species represented by this file.
water   The name of the file, less the ".mop"
-57.8+   The reference heat of formation, in kcal/mol.  If there is no HoF, this part of the line will be blank
1.9+   The reference dipole moment, in Debye.  If there is no dipole, this part of the line will be blank
12.6+   The reference I.P., in volts.  If there is no I.P, this part of the line will be blank
H O   Symbols for all elements in the system
C6H6   The formula for the species represented by this file.
benzener   The name of the file, less the ".mop"  The terminal "r" indicates that the file is a geometry reference
YES   Defines the file as being a geometric reference
H C   Symbols for all elements in the system

For reference data, if there is a reference, then the datum will be followed by a "+" symbol.  Spaces are important - that's why this file should not be edited.
