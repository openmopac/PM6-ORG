Global parameters for use with keyword EXTERNAL

 

Global parameters are parameters that do not apply to any individual element but instead apply to quantities that affect the entire system. 

Current interactions in the PM7 method that involve global parameters: Dispersion energies and forces, individual bond corrections.

Parameter Definition
PAR1 Scalar multiplier for correction of C-C triple bonds, see (1) below.
PAR2 Exponent multiplier for correction of C-C triple bonds, see (1) below.
PAR3 Scalar multiplier for correction of O-H term, see (2) below.
PAR4 Exponent multiplier for correction of C-C triple bonds, see (1) below.
PAR5 Free for use
PAR6 Free for use
PAR7 "s6" in the D3 dispersion for Grimme's method.
PAR8 "alp" in the D3 dispersion for Grimme's method.
PAR9 "rs6" in the D3 dispersion for Grimme's method.
PAR10 - PAR60 Free for use

In method development, the parameters that are "Free for use" can be used in PARAM for optimizing global parameters for old, modified, or new quantities.

Global parameters can be specified in EXTERNAL files as follows:

     ALPB_O      H        2.12
    XFAC_O      H        0.16
    PAR7                 2.28
    PAR8                -0.16
    PAR9                -0.10
    PAR58             -67.89
 
As usual, spacing is not important. The ALPB and XFAC are included in this and the following example to show the style of regular parameters.

In a PARAM run, global parameters can be specified as:

ALPB_O      H
XFAC_O      H  0 2
PAR7  1 3
PAR8 -1 0
PAR9 -1 0
PAR60 0 3
 The quantities following the parameters are the lower and upper bounds allowed.  These are particularly useful as the default bonds are unlikely to be appropriate.
