NOSAVEP

Normally, parameters are saved to the file <filename>.rp whenever a cycle of parameter optimization completes.  If NOSAVEP is present, the parameters will not be saved.  This is useful when experiments are being run, and it is important that the starting values of the parameters do not change from run to run. 

If NOSAVEP is present, then the optimizing parameters will still be printed in the output file, but the message saying, "PARAMETERS DUMPED O.K." will not be printed.
