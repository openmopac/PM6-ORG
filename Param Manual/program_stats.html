PARAM Utility: STATS

 The STATS utility compares the accuracy of various methods. With the exception of the very early stages of method development, it is useful at all stages.  As soon as a new method has reached the stage where it can be compared with existing methods, STATS can be run, and the comparison made. STATS calls the utilities STATISTICS (which does the hard work), and MERGE.

How to use STATS

First, run surveys of the appropriate methods.  Name each survey after the method, so if AM1 is to be surveyed, the data set used in the survey should be called am1.dat or AM1.dat. 
At the DOS prompt, go to the folder that contains the survey files.  These files will have names that end in ".geos", ".heats", ".ips", ".dips".
Assuming the surveys available are AM1, PM3, PM6, and new, use the command:
C:> STATS NEW AM1 PM3 PM6

STATS will then write the message:
What elements? A blank, or 'All', or symbols of elements to be used,
or 'NOT' followed by either 'TM' (to exclude transition metals)
or by symbols of elements that are not to be used.)

Make the appropriate response. 
If statistics for all compounds of H, C, N, and O, only, are to be compared, then type in "H C N O".
If statistics for all compounds are to be compared, then don't type anything or type in the word "ALL".
If statistics for all compounds except those containing the halogens are to be compared, then  type in "NOT F Cl Br I"
If statistics for all compounds except those containing any transition metals are to be compared, then  type in "NOT TM"

Press "return"

Several HTML windows will open - these give details of the statistical analyses. As an example, if PM6 and PM3 are compared for accuracy of prediction of heats of formation for compounds involving H C N and O, only, the table would like like this:

Average Errors in Heat of Formation (kcal/mol )

 

     Element PM6 No. in set PM3 No. in set
  1 Hydrogen 4.42 1123 5.47 1123
  6 Carbon 4.43 1120 5.49 1120
  7 Nitrogen 4.42 463 5.94 463
  8 Oxygen 4.61 626 5.86 626
All elements,
avoiding double counting		 4.63 1154 5.60 1154
Same as above, but
with double counting 		4.46 3332 5.62 3332

 

The color coding allows relative accuracy of methods to be compared. The first method in the list, here PM6, is used as the standard.  1154 compounds involving oxygen (element 8) were used.  For PM6, the average unsigned error was 4.61 kcal/mol.  For PM3, the average was 5.86 kcal/mol.  In other words, PM3 was not as accurate as PM6.  Where the accuracy is lower, the number would be printed in red, where it is higher, the number would be in blue.

The last two lines in the table summarize the statistics.  "All elements, avoiding double counting" counts each system once only. Note that it is possible for the average unsigned error to exceed all of the average unsigned errors of the individual elements. 

"Same as above, but with double counting" counts each compound that contains hydrogen plus each that contains carbon, etc.  In other words, if the statistics for H, C, N, and O were averaged.

Up to four methods can be compared in any one run.  Obviously the lowest number of methods that can be compared is two, but if only one method is used, a set of statistics can still be generated.
